# Recap Generation Instructions

Use this template when generating recaps from transcriptions.

## Format

```
Hey Q fam!
Here is a summary of the latest live stream with @cass_on_mars (@QuilibriumInc founder).

✨ [TITLE]
[Short description - 2 lines max]

▶️ Watch it on YouTube (English captions): [leave empty]
▶️ Watch it on X: [leave empty]


✅ Key topics:
[Short list of key talking points]

[Summary content here]
```

## Formatting Rules

- Use "⦿" for bullet points
- Use "✨" for main title
- Use "✅" for section titles
- Always leave an empty line after titles and paragraphs
- Output as simple text with emojis in a code window

## Tone & Style

- **Tone**: Informative
- **Perspective**: Third person, general statements (not "the speaker says...")
- **Technical content**: Summarize technical details more concisely
- **Non-technical content**: Be more specific and detailed

## Important Notes

- Project name is "Quilibrium" (transcription may contain misspellings)
- Speaker is Cassandra Heart (Quilibrium founder and lead developer)
- Handle: @cass_on_mars / @QuilibriumInc

## Section Structure

1. **Opening** - Standard greeting format
2. **Title** - Extract compelling title from content
3. **Links** - Placeholder for YouTube/X links
4. **Key Topics** - 3-5 bullet points
5. **Summary** - Main content organized by topic
